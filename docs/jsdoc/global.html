MenuItem(name, price, quantity, ingredients) → {html}

Source:
  • components/menuItem/index.js, line 20
Componente que retorna um card com as informações do alimento. Informações fornecidas como parâmetro da função. Possui três tipos diferentes de retorno. Um para quando o cliente acessa, outro para quando o garçom acessa (pode adicionar ao pedido o item) e outro para quando o administrador acessa (pode adicionar, editar ou remover um item).
Parameters:
Name Type Description
name string Nome do alimento.
price number Preço do alimento.
quantity number Quantidade de estoque do alimento.
ingredients string Ingredientes do alimento.
Returns:
Retorna a card com as informações, fornecidas, do alimento.
